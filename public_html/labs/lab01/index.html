What I've learned about HTML so far

Relative File Path
Links files in website from the vantage point of the file you're currently in; accessing a folder within your current one only needs the folder name and then a /, but accessing the folder above requires using .. and a /.
Absolute File Path
Links files in website using path that starts with https://www.domainname.com and then goes into subsequent folders.
